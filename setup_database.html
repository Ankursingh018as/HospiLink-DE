MySQL authentication plugin issue. Please use the manual method below.

Manual Import Steps (Recommended)

  1. Open phpMyAdmin:
    http://localhost/phpmyadmin
  2. Login with your credentials:
    Username: root
    Password: 8511
  3. Create Database:
    • Click "New" on the left sidebar
    • Database name: hospilink
    • Collation: utf8mb4_general_ci
    • Click "Create"
  4. Import SQL File:
    • Select the "hospilink" database
    • Click "Import" tab at the top
    • Click "Choose File"
    • Navigate to: d:\xampp\htdocs\HospiLink\database\hospilink_schema.sql
    • Click "Go" at the bottom
  5. Verify Import:
    • You should see tables like: users, appointments, symptom_keywords, etc.
    • Check that sample data is loaded (users table should have 5 rows)

Alternative: MySQL Command Line

If phpMyAdmin doesn't work, try MySQL command line:

Step 1: Open Command Prompt and navigate to MySQL bin folder:

cd D:\xampp\mysql\bin

Step 2: Login to MySQL:

mysql.exe -u root -p8511

Step 3: Create and use database:

CREATE DATABASE IF NOT EXISTS hospilink;
USE hospilink;
SOURCE d:/xampp/htdocs/HospiLink/database/hospilink_schema.sql;

Step 4: Verify:

SHOW TABLES;
SELECT COUNT(*) FROM users;
After Import Complete:

Access the system at: HospiLink Login
